About Us

The "Player Behavior Prediction for In-Game Purchase Using Machine Learning" project aims to develop a predictive model that can accurately forecast player behavior in relation to in-game purchases. By analyzing player data, such as gameplay patterns, engagement levels, demographics, and historical purchase data, the model seeks to identify key factors that influence a player's likelihood to make in-game purchases.
